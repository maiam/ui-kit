# ui-kit

Um UI Kit leve em **React + Vite + TypeScript + Tailwind CSS**, focado em componentes reutilizáveis e layouts prontos para acelerar novos projetos.

Este repositório nasceu como um laboratório de **reuso de UI** entre vários apps/sites: você copia/instala o kit, e reaproveita layouts e componentes consistentes (com dark mode).

---

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Dark mode via `class` no `<html>` (ThemeProvider + ThemeToggle)

---

## Começando

### 1) Instalar dependências

```bash
npm install
```
